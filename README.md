# 🛍️PHP 쇼핑몰 프로젝트
## 🛠️개발 환경 및 개발 툴
- OS : Linux
- DB : MariaDB
- Tool : PHP, Bootstrap, phpMyAdmin, VSCode
## 🌟주요기능
- 회원 시스템 : 가입, 로그인, 로그아웃, 회원 정보 수정
- 상품 관리 : 상품 목록 조회, 상세 정보 보기
- 주문/결제 : 장바구니 관리, 간소화된 주문 및 결제 프로세스
- 관리자 기능 : 회원 등록/삭제, 상품 등록/수정 등
## 🛠️프로젝트 구조
### 📂폴더
- admin : 관리자 페이지의 파일 및 리소스를 모아둔 폴더
- css : 웹사이트 디자인을 위한 스타일시트(CSS) 폴더
- images : 웹페이지에 사용되는 모든 이미지 파일을 모아둔 폴더
- js : 동적인 기능을 구현하기 위한 자바스크립트(JavaScript) 폴더
- juso : 주소록 검색 또는 관련 기능을 처리하는 파일들이 포함된 폴더
- product : 제품 정보, 등록, 상세 보기 등 제품 관련 기능을 위한 폴더
- webfonts : 웹에서 사용되는 사용자 정의 글꼴 파일이 포함된 폴더
### 📄파일
- cart~.php : 장바구니
- jumun~.php : 주문 기능
- member~.php : 회원 시스템
- order~.php : 주문/배송, 결재 정보
- zipcode : 우편번호
